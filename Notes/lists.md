# Lists

On a fundamental level, lists work like arrays do in Javascript. These are a few examples of lists:

```python
numList = [29, 449, 929]
stringList = ['hey', 'man', 'nice', 'shot']

var1 = 'hello'
var2 = 'world'
varList = [var1, var2]
```

## List Methods

#### `len`: get length

```python
list = [10, 20, 30]
len(list) # 3
```

#### `list`: create a list

```python
list(range(1,5))
list # [1, 2, 3, 4, 5]
```
